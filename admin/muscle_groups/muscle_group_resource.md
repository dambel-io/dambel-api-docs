# Muscle Group Resource

Represents a muscle group in the system.


---

## Schema
| Field              | Type    | Description                  |
|--------------------|---------|------------------------------|
| `id`               | integer | Muscle group ID              |
| `title`            | string  | Muscle group name            |
| `translated_title` | string  | Translated muscle group name |

---

## Example
```json
{
  "id": 123,
  "title": "Abs",
  "translated_title": "..."
}
```
