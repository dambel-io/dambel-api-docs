# Major Resource

Represents a major in the system.


---

## Schema
| Field     | Type    | Description         |
|-----------|---------|---------------------|
| `id`      | integer | Major ID            |
| `title`   | string  | Major name          |

---

## Example
```
{
  "id": 123,
  "title": "Bodybuilding"
}
```
