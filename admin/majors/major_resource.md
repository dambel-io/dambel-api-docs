# Major Resource

Represents a major in the system.


---

## Schema
| Field             | Type    | Description         |
|--------------------|---------|---------------------|
| `id`               | integer | Major ID            |
| `title`            | string  | Major name          |
| `translated_title` | string  | Translated major name |

---

## Example
```json
{
  "id": 123,
  "title": "Bodybuilding",
  "translated_title": "..."
}
```
