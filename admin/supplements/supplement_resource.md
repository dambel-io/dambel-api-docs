# Supplement Resource

Represents a supplement in the system.


---

## Schema
| Field         | Type    | Description                |
|-------------- |---------|----------------------------|
| `id`          | integer | Supplement ID              |
| `title`       | string  | Supplement name            |
| `description` | string  | Description                |
| `link`        | string  | Wiki link (nullable)       |

---

## Example
```
{
  "id": 123,
  "title": "...",
  "description": "...",
  "link": "..."
}
```
