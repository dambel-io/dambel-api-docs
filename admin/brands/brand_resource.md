# Brand Resource

Represents a brand in the system.


---

## Schema
| Field         | Type    | Description                |
|-------------- |---------|----------------------------|
| `id`          | integer | Brand ID                   |
| `title`       | string  | Brand name                 |
| `description` | string  | Description                |
| `link`        | string  | Website link (nullable)    |

---

## Example
```json
{
  "id": 123,
  "title": "...",
  "description": "...",
  "link": "..."
}
```
