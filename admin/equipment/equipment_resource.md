# Equipment Resource

Represents an equipment item in the system.


---

## Schema
| Field         | Type    | Description                |
|-------------- |---------|----------------------------|
| `id`          | integer | Equipment ID               |
| `title`       | string  | Equipment name             |
| `description` | string  | Description                |
| `link`        | string  | Tutorial link (nullable)   |

---

## Example
```json
{
  "id": 123,
  "title": "...",
  "description": "...",
  "link": "..."
}
```
