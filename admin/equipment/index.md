# `GET /api/v1/admin/equipment`
You can get list of the equipment by alphabetical order using this endpoint.


## Response

### 200 OK

```json
{
    "data": [
        {<equipment resource>},
        {<equipment resource>},
        {<equipment resource>},
        {<equipment resource>}
    ],
}
```
