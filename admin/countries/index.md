# `GET /api/v1/admin/countries`
You can get list of the countries by alphabetical order using this endpoint.


## Response

### 200 OK

```json
{
    "data": [
        {<country resource>},
        {<country resource>},
        {<country resource>},
        {<country resource>}
    ],
}
```
