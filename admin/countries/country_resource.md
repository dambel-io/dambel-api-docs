# Country Resource

Represents a country in the system.


---

## Schema
| Field             | Type    | Description         |
|--------------------|---------|---------------------|
| `id`               | integer | Country ID          |
| `name`             | string  | Country name        |
| `translated_name`  | string  | Translated country name |

---

## Example
```json
{
  "id": 123,
  "name": "...",
  "translated_name": "..."
}
```
